# 练习

* 放松肩部手臂：右手从左肩外张至肩膀扩开，眼睛盯大拇指，身体不要转动

* 放松上半身：牛仔和鸭子放松胯部，蹲马步拉琴

* 左右手手型（手指）和手腕保持不变，手肘位置移动。练右手四个平面：1324，左手悬琴弦上方

* 练音准、肌肉姿势：闭眼拉琴

* 左手持琴：约45度，脸转身子不转

* 最好不要触碰琴板，靠近手腕部位和大拇指轻托

* 右手前臂像门绕门轴转动
